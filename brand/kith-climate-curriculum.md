# Kith Climate Curriculum

## Master Claude Code for Climate Work in 6 Weeks

Build real deliverables. Master Claude Code. Solve climate problems fast.

**Next Cohort:** March 16, 2026

**Format:** Live + async support

**Commitment:** 5-7 hours per week

**Enrollment:** Limited to maintain interactivity

**Primary Tool:** Claude Code

---

## What Makes This Different

This isn't another certificate program. You'll build client-ready deliverables using Claude Code and use them to solve real climate problems. By week 6, you'll have a working system, five portfolio-ready outputs, and the ability to solve new problems fast—every time.

**No coding required.** Claude Code handles the technical complexity while you focus on climate solutions.

---

## Weekly Cadence

- **Monday (60 min):** Live build session
- **Tuesday–Thursday:** Independent refinement with async Discord support
- **Friday (90 min):** Demo session—present work, receive feedback, peer learning
